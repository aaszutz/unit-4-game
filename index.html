Buffalo Bills Collector Game

You will be given a random number at the beginning of the game.

There are four buttons below, each button has a value, and by clicking on each button you will be adding values to your total.

You will win the game by matching the value of your buttons, to the random number you are given.

You will lose if the value of your buttons goes above the random number you are given

The value of the buttons are hidden until you click on them.

Each time the game starts, the random number will change ALONG with the values of the buttons!!
